# **App Name**: Firebase Folio

## Core Features:

- Hero Section Management: Admin panel to update name, title, introduction, background visuals, and call-to-action buttons.
- About Section Management: Admin interface to edit bio, skills, and upload/update profile photo.
- Project Management: CRUD operations for projects: title, description, tech used, images, and links (live & GitHub), from the admin panel.
- Service Management: Admin panel to add/edit/delete services, each including an icon, title, and description.
- Contact Form: Form submissions are stored directly in Firestore, with optional email notifications sent to the administrator.
- Admin Authentication: Secure login using Firebase Authentication to protect the admin panel.
- Social Media Links Management: The Admin is able to edit the social media links via the admin panel.

## Style Guidelines:

- Primary color: Dark Indigo (#4B0082) to evoke a sense of professionalism, creativity, and modernity.
- Background color: Light gray (#F0F0F0) for a clean and minimal backdrop.
- Accent color: Purple (#800080) used sparingly for interactive elements and highlights.
- Body text and headline font: 'Inter', sans-serif.
- Code Font: 'Source Code Pro' monospace for code snippets
- Simple, professional icon set (e.g., FontAwesome or Material Icons) to represent services and navigation elements.
- Clean, grid-based layout that adapts fluidly to different screen sizes.
- Subtle animations for page transitions and UI interactions to enhance user experience.
